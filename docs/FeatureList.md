# H5 移动端基础通用框架功能清单

## 📋 项目概述

基于 Vue3 + Vite + Tailwind CSS + TDesign Mobile 的技术栈，构建一套完整的 H5 移动端基础框架。

## 项目框架

- Vue3 + Composition API 核心
- Vite 项目构建工具
- Tailwind CSS 原子化 CSS 框架
- TDesign Mobile 移动端组件库
- vue-router 路由管理
- pinia 状态管理
- axios 网络请求库
- vueuse Composition API 工具库
- lodash-es 工具库
